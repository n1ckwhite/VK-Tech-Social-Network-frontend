# Fontend
## Запускать проект:
 - npm i,
 - npm run start
## Функционал:
-  Регистрация/Авторизация,
- Страница профиля,
- Редактирование профиля,
- Просмотр фото профиля,
- Добавление поста,
- Редактирование поста,
- Добавление лайков,
- Удаление лайков,
- Поиск пользователей,
- Переход на страницу пользователя,
- Добавление/Удаление из друзей,
- Адаптив,
 ## Технологии:
 - Redux,
 - RTK QUERY,
 - React-Router-Dom v5,
 - TypeScript
